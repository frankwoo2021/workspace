技术助力 网友齐心救援洪水受灾民众

2021.07.29

七月下旬，历史性特大暴雨突袭河南，已致逾70人遇难，累计转移安置人数超过百万。灾情牵动世界各地人们的心。人们通过各种方式自发寻找幸存者和他们的亲人。一位网名为Manto的21岁学生，第一时间利用技术帮助洪水中受困的灾民，成为众多救援力量的一个缩影。

Manto是上海财经大学的一名河南籍学生。当她看到河南“千年一遇”超强降雨造成的灾情后，她想要为家乡出一份力。

她回忆道：“我在网上看到各种求助的信息。但是救援队伍处于供不应求的状态，而且信息非常庞杂。我试着寻找一种办法，帮助救援队与求助人员对接起来。”

Manto的学姐在新冠疫情期间组织了一个线上活动，帮助了很多的人。受此启发，她发了一条朋友圈，开启她的救援行动。

Manto在微信朋友圈发布了自己对在线互助文档的最初设想。

“我觉得灾情紧急，可以利用网络的力量迅速收集和传播准确的信息。”Manto说：“我的主要想法是迅速整理、核实网上的各种求救信息，并搜集新的救援联系方式。”

“我当时在想，我们能否先将受灾人员的信息统计一下，核实是否得到救援，然后将没有得到救援的分为强、中、弱三种紧急程度，再匹配救援人员。”

7月20日，暴雨仍在肆虐，Manto在腾讯文档中创建了一个电子表格，她经常使用这种在线共享文档收集信息以及与同学跨团队合作。随后，她将这一想法分享到初高中同学微信群里。

她的提议很快得到同学的响应。率先加入的几位同学是工科男生小黎、Manto的初高中好友小萌和初中班长宗淼。

几小时内，Manto的“河南暴雨远程救援小分队”便诞生了，其中32名成员均为Manto以前和现在的同学以及实习期间认识的同事。

Manto和志愿者通过微信群协作互助。

他们创建的《待救援人员信息》文档很快在网上传播开来，在不到24小时内有超过250万次访问，第二天文档更新了270余次，成功将受灾人员和救援人员连接起来。与此同时，这份文档从一个简单的信息表变成一个多功能的待救援人员信息中心。这份在线文档刷屏后，涌来的信息让小分队难以招架。Manto赶紧联系了专业的救援团体和其他志愿者们，共同对文档进行管理和更新。

现在，虽然距离暴雨袭城已过去一周，但这份文档仍然在救援灾民和提供其他重要信息方面发挥着关键作用。表格内，新的子表单和功能不断出现，包括一张对于受困人员来说非常有用的医疗自救信息表。网友们持续更新自救知识和危急情况处理方案，甚至包括待产孕妇生产指南。医生也积极提供远程医疗咨询服务。

整个文档是去中心化的，信息通过“众包”的形式由志愿者提供。Manto开了个好头，而现在成千上万的爱心人士在没有任何人提出要求的情况下主动添砖加瓦。小小善举彰显的是信任和善意——这正是对救援工作和灾后重建至关重要的精神。

随着这份在线文档走红网络，越来越多的人参与进来。“各行各业的人士利用他们的各种专业知识，使文档更加有效。这是集所有人力量做到的事情。”Manto总结道。

Manto和她的同学退到幕后，帮助更多专业志愿者、官方力量和救援组织更好地利用这个平台，救助更多的人。

这个表格已经更名为“河南暴雨线上综合信息服务平台”。现如今文档有了英文版首页、更加清晰的跳转目录，增加了心理辅导和支援模块，还为暴雨仍然严重的新乡新增了特别版块。救援需求、避险场所和物资援助信息也在增加。

这份在线文档已成为一个救援互助协作平台。

Manto亲眼目睹了互联网的力量，毕业后想成为一名产品经理。她热切期待着发挥互联网的潜力帮助更多有需要的人。

“互联网真的可以用最高效的方式解决很多问题。我希望，我能做出对大家有所帮助、可以让这个世界变得更好的产品。”
