新型冠状病毒肺炎大众防护手册

本手册有助于全球公众对新型冠状病毒肺炎及其防控知识的了解，以提升防控的信心和能力，为全世界打赢疫情防控阻击战做出积极的贡献。

腾讯医典疫情科普

腾讯医典与医学专家合作推出疫情科普文章和视频，为用户提供居家防疫、居家隔离和及时就诊等疫情防控科普信息。

新冠肺炎自筛工具

运用AI技术，根据中国国家卫健委、中国疾控中心及世界卫生组织发布的最新医学指南设计，提供此初步自我评估工具及预防措施建议。（请注意，本工具并不能作为诊断依据或代替医学咨询。）

微医“全球抗疫平台”提供线上健康答疑（非医学诊断）、心理健康支援、防疫科普指导和疫情播报等实时资讯服务。

丁香园面向全球受众提供疫情动态数据实时更新，以及科普、辟谣知识，用户可在第一时间、一站式了解疫情信息，并面向国内外医生推出疫情公开课。

新冠肺炎防治医院方案

医院是此次新冠战疫的前沿阵地，该《方案》着眼于医院面对疫情的总体部署和全局把控，助力全球医疗行业抗击新冠疫情。

“战疫先锋，同洲共济”全球线上学术交流会

由广州呼吸健康研究院主办，GE医疗、腾讯联合承办, 向全世界分享中国对新冠患者筛查、诊断、治疗的全流程诊疗经验。

全球抗击新冠病毒肺炎联盟研讨会

全球抗击新冠病毒肺炎联盟组织系列在线论坛和学术研讨会，搭建全球医疗顶级专家的合作平台，共同分享经验，相互提供技术支持。

助力远程办公

提供云服务产品及技术协助，帮助人们开展远程教育和办公。
