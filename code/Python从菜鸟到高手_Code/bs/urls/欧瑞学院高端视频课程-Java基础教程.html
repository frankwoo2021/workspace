    课程简介

    本课程为Java语言编程基础课程。Java语言编程是目前最流行，也是用户最多的编程语言。Java语言横跨移动、服务端和桌面三个领域。因此，学会使用Java是非常必要的。无论将来从事Android、Java EE，或是桌面程序开发，都离不开她。本课程详细介绍了Java必须掌握的技术以及常用API，包括基本的语法、数组、集合、面向对象技术、网络、线程等。
