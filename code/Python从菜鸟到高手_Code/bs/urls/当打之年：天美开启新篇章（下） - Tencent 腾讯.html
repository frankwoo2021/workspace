当打之年：天美开启新篇章（下）

2021.12.13

我们的全球员工为您讲述如何打造3A游戏工作室 

天美是腾讯游戏旗下精品游戏研发工作室，总部位于深圳，也是多款热门手游的研发商。不到三年间，天美在中国以外筹建了四个工作室，与位于中国的天美总部各个工作室协同研发下一代游戏。

在本文上篇，七位来自全球各地的天美成员分享了自己的成长经历、工作感悟和加入天美的体验。本篇，他们将与读者分享游戏开发行业的发展方向，以及天美如何通过国际协同研发进军3A游戏开发。

在所有人都远程工作的情况下，要如何制作复杂的游戏？这是协同研发模式所面临的挑战之一。但这也是它最大的优势之一。地理和时区都不再构成障碍，你可以找到最适合这份工作的人。文化和背景的融合可以激发出绝妙的创意。制作一款3A游戏需要投入大量时间，依托拥抱多元、海纳百川的工作室文化，协同研发可以加快这一进程。  


Rosi Zagorcheva，Team Kaiju运营总监

游戏是一种极具创意性和技术复杂性的内容，因此我们最大的挑战是完全远程地实现整个制作。然而远程建立新团队、工作室和制作游戏的挑战令我感到兴奋。我坚信“远程优先”将成为我们未来的竞争优势。

对于Team Kaiju来说，分散式工作将成为常态，这是未来的趋势。尽早建立一套有效的流程可以为我们回到办公室后的成功运营奠定基础，因为我们不再局限于只与同一地区的人合作。

Steven Rhee，天美洛杉矶工作室灯光专家

实施稳健的全球人才流动计划可以帮助企业在任何地方找到并聘用最优秀和最有才华的人才，从而扩充人才库，将企业的多元化水平拉升至新高度。

多年来，我拥有与EA上海团队共同开发《疯狂橄榄球》系列游戏的经验。语言始终是一项挑战。不过另一方面，这又带来了优势，亚洲和美国之间的信息与文化共享使得游戏作品更为多样化。我相信分享想法和专业知识会创造新的文化形式。

Alan Xu，天美深圳工作室动画总监

无论是在洛杉矶、新加坡、蒙特利尔、深圳还是上海，你很难仅靠一个地区的一家工作室快速组建一个完整且综合的团队去开发3A游戏。在之前的公司，我参与的每个项目都有来自不同地区和时区的工作室进行协同研发。协同研发是大规模游戏开发的捷径。

首先，当你在中国、加拿大、新加坡或美国都有团队时，你会有更多选择，有机会找到更好的人才。

其次，每个工作室和地区的人才都有自己的文化和思维方式，这可能会限制游戏的多样性和创新性。而多地域、多文化的团队合作更有可能打破这些思维的局限，带来更具创造性的合作。

第三，协同研发有助于分散技术开发的压力。例如，一个工作室可以负责单人剧情部分，一个工作室可以制作多人在线战斗部分，而其他工作室可以负责转换部分的内容、车辆、生物、高精度数字角色等。

Francois Dallaire，天美新加坡工作室技术美术专家

我在上一家公司时，和我们上海的协同研发工作室有12个小时的时差。那时在项目中，作为项目负责人，你可能会在一天中的任何时候遇到问题，包括半夜。所以我只能把手机调成免打扰模式。不能把工作带入睡梦中，这很重要。

你不能把自己的工作方式强加给协同研发的工作室。不同的文化适合不同的环境。有些人更适合在团队中协作，有些人更适合独立完成任务，有些人面对公众会害羞，而有些人面对公众可以表现得非常出色。每次合作都应该从文化培训开始，让每个团队了解行为好坏的标准，以及如何根据不同的背景进行沟通。有时候，你可以在沟通方式上做出一些小小的改变，这些改变对你个人没有影响，但是对你的目标受众会产生很大的影响。

Sinjin Bain，天美洛杉矶工作室执行制作人

天美在全球协作研发方面具有得天独厚的优势。业界普遍认为我们是手游开发领域的佼佼者，这为我们招揽世界级人才打下良好的基础。我们看到北美地区顶尖的创意、艺术和技术人才欣然加入天美，共同开发全球跨平台3A互动体验。

Alan Xu，天美深圳工作室动画总监

3A游戏开发是一个从慢到快的过程。一个经验丰富的3A游戏团队可能需要一至两年的时间来完成概念和创意的探索，用二至三年的时间来完成前期制作。所以，用四五年的时间开发一款游戏根本不足为奇。但是在这之后会有一段神奇的时期。

通常，80%的游戏内容是在最后一年到一年半的时间内完成的。每一款3A游戏的开发都要经历漫长的过程，在此期间，耐心、信心和决心都是必不可少的。我在天美看到了这三个特质。

Francois Dallaire，天美新加坡工作室技术美术专家

在天美，我从一开始就获得充分的信任，他们相信我会选择最好的方式来构建管线。我的职责是确保美术总监能够实现他的游戏愿景，同时充分考虑当下行业的技术条件。在尊重这些技术条件的前提下，你有充分的创作自由。

Gilles Beloeil，天美蒙特利尔工作室概念美术专家

我的目标是制作出符合创意团队想法的图像。因此我会谨慎选择我要参与的项目。天美蒙特利尔工作室想要制作的游戏鼓舞了我，它有潜力成为一款出色的游戏。这是我加入团队的主要原因之一。

我们之所以从事这个行业，是因为我们喜欢创造世界和故事。创作空间越自由，我们就越快乐。当然，我们的最终目的是为工作室创造优秀作品，所以遵循一些指导原则是十分正常的。

Alan Xu，天美深圳工作室动画总监

3A游戏是一个非常复杂的集合体。环境渲染、角色建模、游戏玩法、电影叙事、特效、音乐等，每个模块都需要大量的技术管线支撑才能打磨出高品质。我不指望仅凭过去的经验就能重新创造一套行业流程。我追求的是建立更高效的高质量管线。依托天美的实力和开放性，如今我获得了这样的机会。

Rosi Zagorcheva，Team Kaiju运营总监

在游戏制作领域，真正能够帮助个人和团队脱颖而出的是能够坦然面对不确定性，并愿意承担创新的风险。游戏是艺术与技术融汇的复杂产物。它们寻求创造全新的、前所未见的体验。要做到这一点，需要勇于尝试，有时会失败，但是可以从失败中吸取教训，再接再厉。

Rosi Zagorcheva，Team Kaiju运营总监

当我在工作中遇到挑战时，我通常会做三件事：

1. 专注于我想要得到的结果。

2. 竭尽全力工作，消除我对失败的恐惧。

3. 暂停几分钟，重新集中精力，以此缓解压力，提高我的决策能力。

要打造一款真正成功的游戏，通常需要经过数年的失败和缓慢的改进。不过，即使我们的大部分尝试都失败了，只要我们仍然能想出一些经过验证的好主意和创新，最终仍然能够获得玩家的认可。

Rosi Zagorcheva，Team Kaiju运营总监

我们从一开始就设定了明确的使命：打造具有突破意义的游戏，形成充满活力的玩家和粉丝社群。我们不断演进优化策略，让所有人参与工作室的建设。但好的策略离不开强大的文化。事实上，企业文化引领商业战略——文化是人们充分发挥潜力进行合作的必要条件。

（对团队而言）最重要的是心理安全。让大家大胆表达想法，没有后顾之忧。这能激发创造力，消除对失败的恐惧。如果我们所有人都能表达真实的想法，我们就能促进创新和创意，这对业务而言是一个巨大的助力。

首先要把多样性作为一个业务目标，我们要努力创造良好环境来鼓励并赞扬多元化的体验和观点。

Derek Yao，天美工作室群HRBP

我们投入了大量精力来优化管理，促进全球研发网络顺利运行。首先要做到的是包容和开放。我们充分尊重国外的风土人情，在主要战略区域的工作室设置了本地运营和人力资源团队。

在内容产业中，游戏是技术和创意的完美结合。一个团队可以同时包含有才华的人、有极客精神的人，还有实用主义者。虽然背景和专业各不相同，但是他们聚在一起可以创造出有趣的、吸引游戏玩家的内容。

 

咖啡、午睡、家庭时间——这是天美同学工作日的一部分。当然，也有站立会议、一对一谈话、严肃的讨论以及解决各种问题。不过如今，办公地点可能就是你家，你可以根据自己的需要灵活安排一天的工作。

Francois Dallaire，天美新加坡工作室技术美术专家

我在家办公，所以……早上八点半起床，穿上最舒适的短裤，冲一杯咖啡。

上午九点，我会查看我的任务，与团队成员沟通，开始埋头工作。

中午十二点半，我和女儿一起吃午饭，然后我们睡个午觉。

下午一点半，再喝点咖啡，然后继续工作。

下午六点与家人一起吃晚餐。

下午六点半到七点半，整理没有完成的任务或准备材料，为第二天做好准备。

Rosi Zagorcheva，Team Kaiju运营总监

因为我们团队规模不大，而且在不断扩展，所以我每天的工作内容往往取决于团队当前的关注点。一般来说，我在规划工作内容的时候会有明确的目标，那就是提高自己和团队的工作效率。

我通常会在早上专门拿出时间去做一些有深度、需要专注的工作，随后，我们团队会有好几个小时一起就关键任务进行合作。我们每天早上都有站立会议，而且要进行游戏测试，因为我们认为每天都要玩自己的游戏才行。每到周五，我们还会玩其他游戏来寻找灵感和创意。

Gilles Beloeil，天美蒙特利尔工作室概念美术专家

我的工作日是从查收和回复电子邮件开始。然后开始完成我的任务。大部分时间我都是在Photoshop中画画，但有时我也会专注于构建3D内容或寻找视觉参考。下班之前，我会再次查看电子邮件。

Steven Rhee，天美洛杉矶工作室灯光专家

为了提高工作效率，我早上会先喝一杯美味的咖啡，之后安排必要的任务，并与我的海外同事和深圳团队讨论工作进展。

Alan Xu，天美深圳工作室动画总监

我每天早上都和团队开会，确定每个人当天的工作重点，然后针对具体问题与其他部门联系，确保我们保持一致。下午我会花一至两个小时自己制作动画，以及查看团队的工作情况。每天我会与一位团队成员进行一对一谈话。下班前，我会留出半小时到一个小时的时间梳理想法，思考如何优化现有的工作流程，探索新的风格和创意。

Rosi Zagorcheva，Team Kaiju运营总监

我喜欢到户外去。可以感受到更广阔的世界，超越我本身，超越我所面临的问题，这能给我带来新的视角和内心的平静。

Gilles Beloeil，天美蒙特利尔工作室概念美术专家

我喜欢运动，喜欢和我的孩子们一起玩，喜欢看书、看电影、玩电子游戏和冥想，不过有时候根本没办法进行冥想，因为家里有两个孩子。

我教授CGMA场景绘画和设计已经有六年了。在批改学生作业的过程中，能让我理解哪些东西对于绘画而言是重要的，对我自己也是很好的修炼。

Steven Rhee，天美洛杉矶工作室灯光专家

我喜欢跟我的女儿和妻子待在一起，喜欢摄影、玩游戏，还有在周末和朋友一起打高尔夫。

Alan Xu，天美深圳工作室动画总监

我会读小说，然后尝试把小说中的一些场景想象成图片。

Francois Dallaire，天美新加坡工作室技术美术专家

闲暇时我喜欢看动漫剧集，和家人一起在游泳池里畅游，出去尝试没吃过的食物和餐厅。

Francois Dallaire，天美新加坡工作室技术美术专家

数据科学的发展将对技术美术使用的工具产生重大影响。通过使用机器学习或相关技术，我们可以越来越多地让曾经不可能的任务实现自动化。它们可用于生成过程化内容，比如通过算法去生成侵蚀地形，或植被等美术物料的自动填充，以及通过面部扫描模型去自动化生成角色的基于骨骼的形变数据，产生面部变化……这类应用范围仍在不断拓宽。

Gilles Beloeil，天美蒙特利尔工作室概念美术专家

在任何项目中，概念美术都是非常重要的，制作过程中必然需要一些概念美术。如今，我们掌握的工具更加高效，而且世界各地有很多很棒的概念美术学校。所以，作品的质量水平将继续提高。

概念美术将越来越多地应用虚拟现实（VR）技术。

Steven Rhee，天美洛杉矶工作室灯光专家

下一代游戏开发是游戏行业最重要的话题。Epic虚幻引擎5中的Lumen和Nanite两种新技术带来的革新，会让下一代游戏开发者发挥出无限想象力。新技术的出现是一个很好的机会，开发人员可以在下一代游戏中呈现过去无法实现的内容。

Alan Xu，天美深圳工作室动画总监

在动画方面，继动作捕捉技术之后，多年来一直没有出现革命性的技术更新。直到两三年前，才出现基于AI学习的动画生成技术。腾讯的AI技术积累和人才储备远远高于一般游戏制造商。

我们目前正在研究基于AI学习的各种动画制作技术解决方案，包括角色肢体动作、面部表情、物理碰撞模拟等。我认为这是未来大规模游戏动作制作的必然趋势。

Francois Dallaire，天美新加坡工作室技术美术专家

不断了解新技术，永远保持好奇心。多接触同领域内充满激情和才华的人。

Gilles Beloeil，天美蒙特利尔工作室概念美术专家

首先，你必须有令人信服且内容广泛的作品集。你可以展示传统绘画、照片、数字作品等。这是吸引美术总监注意的关键。

对每件事、每个人都保持好奇心！

Steven Rhee，天美洛杉矶工作室灯光专家

走出去，观察这个世界，而不是只坐在电脑前。真实的世界是灯光和现实主义的绝佳参考。你自己的体验就是最好的老师。

建立自己的视觉库。对光在现实生活中的作用和机制保持好奇。还有学习摄影。

不断向自己提问。例如，“天空为什么是蓝色的？”或者“日落和日出为什么有如此美丽、温暖的颜色？”理解最简单的光影，对创造好的灯光效果将有很大有帮助。

Alan Xu，天美深圳工作室动画总监

作为动画领域从业者，我收集了很多电影和动画资料。然后我会把它编辑成片段，在业余时间逐帧观察学习。
