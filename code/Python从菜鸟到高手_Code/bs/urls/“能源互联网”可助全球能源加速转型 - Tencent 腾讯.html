“能源互联网”可助全球能源加速转型

2021.08.20

网大为（David Wallerstein）作为腾讯首席探索官，在过去几年间与众多科学家、研究机构、行业和组织广泛合作，专注于促进可持续发展，加快能源转型，努力实现碳中和。

近期，网大为受邀出席“国际能源变革对话”，并就如何发挥包括云、软件、数据、人工智能和机器学习等互联网先进科技在能源转型中的重要作用，分享了前瞻性的观点。

“国际能源变革对话”由中国国家能源局和国际可再生能源署共同主办。参会嘉宾包括：国际能源署署长Fatih Birol博士、国际可再生能源署总干事弗朗西斯科·拉卡梅拉（Francesco La Camera）先生，以及其他致力推动可持续发展的领袖和专家。

下文摘自网大为的论坛演讲，部分内容略有删改。

观看演讲视频了解更多关于能源转型的概念性框架。

随着能源生产和存储的大幅增加与分布化，以及电力消费者的增长，我们面临着更加复杂和严峻的挑战。然而，这亦是提升能效，优先消纳绿色能源的重要机遇。

我们可以重构市场，激励最有效的策略，通过降低电力需求、平衡峰值，减少整体资源浪费并提高生产力。

我相信我们面临一个重大的机遇，可以借助软件、数据、云计算和人工智能（AI），重构能源生态系统。我们甚至可以称之为“能源互联网”（IoE），不久的将来，这个概念就能变成现实。

多年来，人们一直在探讨“软件层”对能源市场的创新潜力。软件层既涉及市场规则，如定价和税收，也包括贯穿整个能源系统的软件，它们能够分析和预测，甚至完成智能操控水泵或空调系统等任务。

软件、AI和其他能源互联网创新使这一切成为可能。

软件汇聚一切力量

让我们暂时远离混乱的现实世界，毕竟解决实际问题十分复杂，而且耗费心神。

我们可以试想一下：能源价格是完全动态的、实时的，对增量参与者开放，他们可以将自己的能源出售给电网。此类能源的定价可以准确反映出其“绿色度”（即二氧化碳份额）、能源生产的效率和充裕程度。假设某天，某地区的风力发电机组满负荷运行，就可以推测出当天的风电价格达到低点，而火电价格则达到顶点。

如果我们能够建立这种透明的市场定价制度，我们就可以为不同场景和行业中AI驱动的第三方能源优化程序创造一个生态系统。

以软件解决方案优化能源生态系统

每个客户，从海水淡化厂到电动车车主，都可以在优化的电价机制中平等受益，这反过来也会反映出能源的绿色度、丰富度、边际成本等其他因素。

让我们更进一步，设想一下这个激动人心的场景：未来能联网的各种能源消费设备，可以推动用电需求与能源目标和价格相同步。试想，建筑冷却系统与绿色发电系统协调同步；农田里的水泵、甚至电动车、热水器或空调，都能够与电网协同工作，满足供暖、冷却和充电需求。这种方式不仅可以优化电价，还可以减少浪费和排放，有助于最大限度地发挥现有基础设施的价值。

我们也不要忘记，在某些情况下，能源消费者也有可能成为能源供应者。当需要储存能源时，未来的电动车也能以合适的价格向电网供电。

为速度而生的“能源互联网”

随着能源转型加速，我们需要考虑如何更多地利用能源互联网。关键的挑战在于找到并启动正确的项目，以及协同所有参与者以加快进展。

在过去的几十年里，我们认识到软件的运行速度堪比光速。它可以高效运行，快速驱动生产力，帮助我们每天不断探索和进步。

就紧迫性而言，我们能源生态系统中的这个“软件层”可以与现有硬件（如工厂和设备）一起工作，此外配备传感器的数据流和执行元件或许还可以降低成本。因此我相信，这些AI解决方案能够比新建基础设施或进行硬件升级更迅捷。基于这一点，我们又可以利用越来越多的软件驱动途径来加速能源转型。

能源存在于广泛的生态系统中，以满足人类的基本需求。电力行业是水资源密集型产业。农业亦是完全依赖水资源。在美国等大型发达国家，农业耗水量约占40%，另外还有40%用于电力系统。

透过“FEW”看问题

如果从食物、能源和水（FEW）这三个方面来解决我们的能源转型需求，实现能源转型将更有希望。

通常情况下，升级为光伏或风电可以减少对水资源的依赖。对于水资源紧张的地区，这可以启发人们思考改善水资源利用的新途径。

AI有助于实现高效的水资源利用策略。对于室外农业，一些AI策略可以节省高达40%的用水量，同时通过精准、智能的方式促进农作物健康和产量。同时，抽水需要耗费大量能源，因此只需改善作物健康状况，我们就可以节省大量能源和水资源。

反之，即使在农业生产中使用可再生能源，也不代表我们可以把大量的水抽取到农田中。AI可以帮我们反复确认优化方向，例如作物健康，AI能帮助我们找到节省宝贵的FEW资源的策略。

前述几项IoE驱动的策略结合采纳效用最大。在未来，透过软件和数据，许多优化系统可以相互交流，协同工作以最大限度地提高效率和实现优化。AI、软件和云技术将协助我们节约能源，最大限度地利用绿色能源，节约用水，提高农业和工业生产力。

但最重要的，是在充满挑战的情况下提高人类的生活质量。

我们希望能够与大家多多交流，继续探索这一问题，共同创造更美好的能源未来，合作应对关键挑战。
