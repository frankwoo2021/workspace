“天才妈妈”以祖传工艺打造潮流时尚

2021.09.02

当绵延千年的传统工艺与最潮时尚相遇，会碰撞出怎样的火花呢？

答案无需猜想，因为这已经在中国的偏远地区发生：一群技艺超群的女性已抓住机遇，大放异彩。

“天才妈妈”是腾讯公益慈善基金会为支持乡村妇女发展而设立的项目。相比大都市的女性，乡村妇女往往更缺乏发挥所长的机会。

该项目为中国偏远山区和乡村妇女筹集资金，让她们得以施展传统技艺，传承宝贵的文化习俗和非物质文化遗产。这一善举不仅可以改善她们的生活，更帮助她们学习利用数字工具和新的方式表达自我，发挥创意。

最近举行的一场时装展便是项目的绝佳案例。

今年上半年，九位来自偏远地区的时尚绣娘远赴上海，与时尚设计师和明星合作，利用传统工艺打造了一场夺目的T台秀。设计师们将民族传统绣片与现代电脑印花相结合，展现出一种富有生命力、大胆创新的时尚风格。

“上海时装周X约会天才妈妈”活动在上海静安嘉里中心举办。

苗族天才妈妈

当你走进上海时装周展厅，你会发现当中伫立着一件格外显眼的红色礼服，旁边有一位苗族姑娘正在描绘。她叫王欣颖，来自贵州丹寨。她正在进行“蜡染”，这是一门源自2000多年前的印染技艺，而苗族的祖先和长辈便是通过这门技艺来表达自己的美术风格、审美与想象力。

王欣颖展示贵州丹寨腊染。

一只蜡染而成的鸟，就充分体现当中所蕴含的苗族文化——飞鸟寄托着对美好生活的憧憬与希冀；鱼鳞代表着对多子多孙的祈盼；鸟的翅膀却由蝴蝶翅膀演化而成，这可追溯到苗族的起源，传说蝴蝶妈妈孕育了世间万物。

蜡染中的螺旋纹、鸟纹、鱼纹。

王欣颖在家乡的生活十分忙碌，她是两个孩子的母亲，同时也要兼顾自己的蜡染工坊。借助跨界合作和多媒体推广，她的事业得以蓬勃发展。

大忙人妈妈

王欣颖并不是唯一一个得到“天才妈妈”项目帮助的人。何茂晴──一位来自贵州、笑容爽朗的妈妈──同样获得了项目的帮助。

她说：“以前我没有工作，现在我是个大忙人啦！”

何茂晴传承着扎染手艺。年轻时，村里穷困，为了赚钱和拓宽视野，她走出大山，来到广州打工，一晃就是十几年。由于不想让女儿成为留守儿童，她最终选择回到家乡。

何茂晴向我们介绍贵州荔波扎染。

扎染不仅是她补贴家用的技艺，更是她的快乐源泉。她说：“在全情投入扎染时，我便能享受它带来的快乐。”我曾经认为自己一无所长，但现在我享受到做扎染所带来的成就感。

“我来到上海向更多人介绍和展示我们的传统文化，我感到很自豪。”

 何茂晴亲手做的扎染作品，生动可爱。

在家乡，何茂晴经常把村里的妇女聚集起来，一边做扎染，一边帮她们接订单、售卖成品，实实在在增加了大家的经济收入。扎染成为一股凝聚人心的力量，让村里的人情味越发浓了。

农民画妈妈

你可能没有听过位于吉林省的东丰县，但当地有着极为宝贵的非物质文化遗产──农民画。东丰县农民画历史悠久，以色彩亮丽、构图饱满、乡土风味浓厚而独树一格。

刘莉高中毕业后因家庭条件所限被迫放弃学业，跟随父亲学习农民画，至今其画作已被多本画册收录出版，使她能够赚取额外收入并继续学业。她介绍说：“农民画出自农民之手，能够真正展现当代农民在生产过程中的情感和状态，是一种现实主义的艺术展现。”

受诸多因素影响，东丰县很多女性都未能找到合适的就业出路。东丰农民画约400位骨干创作者中，像刘莉这样的女性画家占了三分之二以上。

东丰农民画至今已有580件作品获奖，先后被送到瑞士、阿根廷和新加坡等国展出，近800件作品被世界知名美术馆收藏，还被送往联合国总部举办画展。得知在家乡也可以赚取可观的收入，很多外出打工的女性也回到了东丰从事相关工作。

绽放乡村“她力量”

自《北京宣言和行动纲要》制定以来，女性平权进展仍相对缓慢。直至今日，全球适龄就业妇女进入劳动力市场的比例仍不足50%。在世界许多地区，女性接受教育的机会依旧不足。而众所周知，无论在公共或私营机构，女性领导者仍然凤毛麟角。

性别平等是可持续发展不可或缺的一环。在中国乃至全世界，保持开放态度并确保女性获得同等的发展机会极为重要──这也是“天才妈妈”项目的目标之一。

得益于互联网的大力推动，“天才妈妈”和腾讯公益慈善基金会得以筹集资金，赋予众多像王欣颖、何茂晴和刘莉这样的乡村女性一展所长的机会，让她们施展才华和创意，走向充满无限可能的广阔天地。

天才妈妈们的蜕变，让我们看到数字化公益扶持女性发展和乡村振兴的潜力。更多公益组织可以运用数字化技术提升专业度及影响力，更好地应对乡村振兴中的机遇与挑战。
